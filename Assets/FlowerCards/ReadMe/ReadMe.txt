Thank you for purchasing our 'Flower Cards' pack.
This package contains a complete set of flower cards ('Hwatu' in Korean). 
If you plan on creating a card game and you don't want to waste time designing the cards, this package is for you. 

< Description >

- This standard 48-card deck can be used for games such as GoStop, MatGo, SeotDa...
- Also contains 3 bonus cards.
- Optimized for HD games.
- These cards are suited for Korean games. If you need Japanese-style flower cards, please contact us.

< Features >

- There are only 4 tris in each card.
- All card images are inside a single 2048 x 2048 pixel texture.
- Only 2 'draw calls' regardless of how many cards are used.

< Structure >

- Each prefab contains front and back sides.
- All the sprites were made using the 'grid slice' feature in Unity 2D's sprite multiple mode.

< Instructions >

- Drag the card prefabs from the 'Prefabs' folder to 'Hierarchy' view or use 'instantiate' function from your script.
- The 3 bonus cards are to be used or not depending on your game's rules. 

< Testing >

- We have successfully completed a test in Unity 3D editor.
- Open the "FlowerCards" scene file in the 'Scene' folder.
